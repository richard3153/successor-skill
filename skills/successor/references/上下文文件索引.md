# 上下文文件索引

## 优先级列表

| 优先级 | 文件 | 用途 | 关键信息 |
|--------|------|------|----------|
| P0 | SOUL.md | 身份定义 | 智能体的人格与行为规范 |
| P0 | MEMORY.md | 长期记忆 | 重要决策、项目背景、用户偏好 |
| P1 | memory/YYYY-MM-DD.md | 近期日志 | 最近几天发生的事件 |
| P1 | AGENTS.md | 工作区说明 | 目录结构、工具配置 |

`
1. SOUL.md       # 必须首先读取
2. USER.md       # 了解服务对象
3. MEMORY.md     # 了解长期背景
4. HEARTBEAT.md  # 检查是否有紧急任务
5. memory/*.md   # 按日期倒序读取最近的日志
6. AGENTS.md     # 了解工作区
7. TOOLS.md      # 了解工具配置
`

## 上下文断裂检测

如果发现以下情况，说明上下文可能不完整：
- 任务描述在 MEMORY.md 中有记录，但工作区没有对应文件
- 最近日志中有「进行中」的任务，但没有进展记录
- 有周期性任务但没有执行记录
